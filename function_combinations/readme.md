## Student Task

Write only the main() function. Don't change any function!

Your main() must:

1. Ask the user for a username using get_non_empty_input

2. Ask the user for a password using get_non_empty_input

3. If either value is None → print an error and stop

4. Authenticate the user using authenticate

5. If authentication fails → print an error and stop

6. Ask the user for a message using get_non_empty_input

7. Validate the message using is_message_allowed

8. If the message is invalid → print an error and stop

9. Encrypt the message using encrypt_message

10. Format and print the result using format_encrypted_output
